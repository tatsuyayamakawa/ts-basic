# TypeScript Basic
This repository is a template for TypeScript programming.

## Project setup
```
npm install
```
```
yarn install
```

## How to use

Launch a local server
```
npm run start
```
```
yarn run start
```

Build
```
npm run build
```
```
yarn run build
```

Start analysis tool
```
npm run lint-fix
```
```
yarn run lint-fix
```
